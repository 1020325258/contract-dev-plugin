# Sales Project Plugins

销售项目 Claude Code 插件集合，提供专业的开发辅助工具。

## 插件列表

### Contract Dev Plugin

签约系统代码开发辅助插件，提供代码审查、数据流追踪和开发规范指导功能。

**核心功能**:
- `/code-review` - 自动对当前分支修改的代码进行 CR 检查
- `/dev-help` - 基于签约系统的开发规范辅助开发
- `/trace-data-source` - 追踪指定变量或方法的来源
- `/create-code-knowledge-skill` - 创建签约领域知识 skill 文档

[查看详细文档](./contract-dev-plugin/README.md)

## 职责

为销售相关项目提供代码开发辅助能力，包括代码质量保障、开发规范指导和业务知识沉淀。
